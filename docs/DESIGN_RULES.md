# Design rules (learned from the user — apply automatically)

1. **Curved arc = a door.** Wherever a plan shows a quarter-circle swing arc, place a
   DOOR in the opening (the gap) at that arc. The arc marks the door + its swing.
2. **Doors:** model the door leaf; default state CLOSED (in the opening) unless told open.
3. **Bed:** queen-size standard (1.53 m x 2.03 m) across all bedrooms.
4. **Toilet:** exactly ONE glass partition, between the WC (toilet) zone and the bathing/
   shower zone. Bathing/shower is the wet zone behind that single glass partition.
5. **Walls align along the bedroom wall** — attached rooms (toilet) sit flush along the
   bedroom's shared wall, walls in line.
6. Entry door of the bedroom = front wall, right corner. Toilet door = middle of right wall.
7. Build rough first, then refine on the user's feedback (parse + correct loop).
